ClarityBot: AI Mental Wellness Assistant

ClarityBot is an AI-powered mental wellness coach designed to provide immediate, empathetic support and guidance to users, particularly students and young adults facing stress and anxiety. It combines a conversational interface with a powerful large language model (LLM) to offer coping strategies and non-medical emotional support in a non-judgmental space. It was built to address the critical need for accessible, non-stigmated mental health guidance.

• Technology Stack: Python, Streamlit (Frontend/UI), Google Gemini API (gemini-2.5-flash-preview-05-20 Model Engine), Requests (Robust API handling).
• Secure Architecture: Implemented Streamlit Secrets for secure API key management, ensuring credentials are never exposed in the public GitHub repository.
• Empathy Engineering: Utilizes a strict system prompt to define the model's persona, guaranteeing compassionate responses while avoiding medical advice or crisis intervention.
• Scalable Deployment: Deployed on Streamlit Community Cloud for permanent, 24/7 global accessibility.
